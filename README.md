# Algorithms

[Algorithms by Robert Sedgewick and Kevin Wayne (4th edition)](https://algs4.cs.princeton.edu/home/) exercises in Java.

## Build project

```bash
./mvnw clean package
```

## Run tests

```bash
./mvnw clean test
```
